### 기능 목록
- 입력
  - 다리 길이 입력
  - (라운드마다) 이동할 칸 입력
  - 게임 재시작 여부 

- 입력값 검사
  - 다리 길이
    - 3 이상 20 이하 숫자
    - [ERROR] 다리 길이는 3부터 20 사이의 숫자여야 합니다.
  - 이동할 칸 
    - 'U' or 'D'
  - 게임 재시작 여부
    - 'R' or 'Q'

- 출력 
  - 게임 시작 알림 문구
    - "다리 건너기 게임을 시작합니다."
  - 다리 길이 입력 요청 문구
    - "다리의 길이를 입력해주세요."
  - 이동 칸 요청 문구
    - "이동할 칸을 선택해주세요. (위: U, 아래: D)"
  - 재시도 여부 요청 문구 
    - "게임을 다시 시도할지 여부를 입력해주세요. (재시도: R, 종료: Q)"
  - 게임 종료 문구
    - 최종 게임 결과   
      [ O |   |   ]   
      [   | O | O ]   

      게임 성공 여부: 성공   
      총 시도한 횟수: 2
